# Component Dropdown 

React app with scoll infinito using intersection observer

# Database

Firestore

## Install

```
npm install or yarn

```

## Run for development

```
npm run start
yarn start
```

## Build to production

```
npm run build
yarn build
```


## Docker run

```
docker-compose -f docker-compose.yml up
running in http://localhost:3000/
```
